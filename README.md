# API de Cadastro de Funcionários

## 📌 Descrição
API REST para gerenciamento de funcionários.

## 🚀 Tecnologias
- Java 17
- Spring Boot
- PostgreSQL
- Docker
- Swagger

## 🧠 Funcionalidades
- Criar funcionário
- Listar funcionários
- Atualizar dados
- Deletar funcionário

## 🏗️ Arquitetura
- Controller
- Service
- Repository
